zabbix-twilioとは
========================================================

zabbixからTwilioを使用して電話による自動通知を行い、 イベントの 障害対応コメントへ記録を残すことができます。

更新履歴
==========

* 2014/04/18 ver.1.0
* 2014/07/01 ver.1.1 zabbix-twilio.phpのログ出力不具合修正

動作環境
========

zabbix-twilio を使用するには以下のソフトウェアが必要です。

* Zabbix 2.0 以上
* PHPが動作するWebサーバー(PHP 5.3以上)
